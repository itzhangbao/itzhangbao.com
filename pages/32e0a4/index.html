Mac 砸壳工具frida-ios-dump使用（动态砸壳）

# 安装Homebrew (Homebrew国内源)

-----------------------------安装 homebrew 开始------------------------------

现在可以直接用 cunkai 的 https://gitee.com/cunkai/HomebrewCN/

Mac 安装脚本：（推荐 优点全面 缺点慢一点）

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
1

Mac 卸载脚本：

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/HomebrewUninstall.sh)"
1

-----------------------------安装 homebrew 结束------------------------------

# 使用 brew 安装 python

-----------------------------安装 python 开始------------------------------

查询 python3 所有有效版本

brew search python3
1

我这里已经安装了python@3.9 所以后面会有对号

安装

brew install python@3.9
1

修改 python3 别名

由于 Xcode 需要使用系统自带的 python2.7 所以系统的 python 别名

终端使用的是 bash：

vim .bash_profile
1

终端使用的是zsh：

vim .zprofile
1

添加：

# Python
alias python2='/System/Library/Frameworks/Python.framework/Versions/2.7/bin/python2.7'
alias python3='/usr/local/Cellar/python@3.9/3.9.4/bin/python3.9'
alias python=python3
1
2
3
4

生效：

source .bash_profile
1

或

source .zprofile
1

验证安装结果

----------------------------安装 python 结束------------------------------

# 安装 frida

手机安装：

cydia添加：https://build.frida.re/ 源，并下载安装 frida

Mac 安装：

sudo pip3 install frida-tools
1

确保手机和Mac上安装的 frida 版本保持一致

安装usbmuxd与手机通信

brew install usbmuxd
1

下载 frida-ios-dump 脚本

下载：frida-ios-dump 并保存到 opt/dump/ 下：

git colne https://github.com/AloneMonkey/frida-ios-dump.git /opt/dump/frida-ios-dump-master
1

安装 dump.py 脚本的依赖：

cd /opt/dump/frida-ios-dump
sudo pip3 install -r requirements.txt --upgrade
1
2

添加 dump.py 别名：

终端使用的是 bash：

vim .bash_profile
1

终端使用的是zsh：

vim .zprofile
1

添加：

# frida-dump
alias frida-dump="python3 /opt/dump/frida-ios-dump-master/dump.py"
1
2

生效：

source .bash_profile
1

或

source .zprofile
1

这里注意： 网络上的陈旧资料都是 alias dump.py="/opt/dump/frida-ios-dump-master/dump.py" ，但是实际终端调用 dump.py 时会默认调用系统2.7版本的python，会报缺少 frida库的错误。

使用：查看手机进程列表

 frida-dump -l
1

# 砸壳

通过USB使用ssh连接越狱设备，将22映射到电脑上的2222端口

iproxy 2222 22
1

另开终端加入连接 密码为设备的登录密码alpine，需要保持连接

ssh -p 2222 root@127.0.0.1
1

或

ssh root@127.0.0.1 -p 2222
1

再开终端开始砸壳你想要分析的App，命令如下：

查看手机进程

 frida-dump -l
1

砸壳：

frida-dump #pid# 或 #name# 或 #identifier# -o #输出ipa的路径，默认当前路径#
1

以今日头条为例：

手机打开日头条

frida-dump 今日头条 -o Desktop/News.ipa
1
